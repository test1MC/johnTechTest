How To Use Analytic Modules

Populating Data

Data Filters house your institution’s unique portfolio segmentation dimensions.

Use the multi-select drop-downs to narrow down the portfolio to a specific product, dimension, cell, etc.

Use to identify “what part of my portfolio do I want to analyze?” Click Apply Filters to apply changes.

Filters can be hidden if desired, to enable a maximum amount of screen real estate to be available for the data visualizations.

The filter label bubbles identify the Data Filters selected, for transparency. Click the x within any bubble to clear the filter and reset the corresponding filter selection to All.

The Chart format drop-downs options detail which metric is displayed for the selected segment of the portfolio, and how that data is organized and displayed.

Use to identify “What do I want to know about this piece of the portfolio?”. All changes result in live updates.

 

Using Interactive Charts

On most Analytic modules a Table or Chart tab is available to view the numeric data that underlies the chart visualization.

If a module has both Table and Chart tabs, click the Table tab to access additional information not visible in the chart.

Click and drag within the chart area on any visualization to zoom into a specific range of X and Y data.

Use the top slider on the chart Y-axis to pull down and zoom into a specific range.

Use the X/Y corner slider to pull either up or to the right to zoom into a specific range.

Use the right slider on the chart X-axis to drag right and zoom into a specific range.

All legends are interactive. Click on an individual series to add or remove it from the chart.

The All or None buttons provide the ability to make bulk selections or deselections.

 

Universal Functionality

The PriceTek logo acts as a "home” button, returning at any time to the login screen.

Download options include:

  • PPT: Screenshot of chart, with headers, in PowerPoint
  • PDF: Screenshot of chart, with headers, as a PDF
  • Excel: Formatted, interactive chart and table data in XLSX format
  • Excel Raw: Raw, unformatted data only, in XLSX format

The exception is CDA Benchmarking and the Beta Dashboard. For these modules, you can download PDF, Image, or Data (in CSV format, which can be open in Excel easily or pulled into a database app).

Note: If there are multiple visualizations present in a module (common for CDA Benchmarking), then the last visualization (i.e., graph) is the one downloaded. To download alternate graphs, click it to select before downloading.

Favorites enable users to save any set of Data Filter and Chart Format Drop-down selections as a Favorite.

Favorites can be saved for quick future reference, used in Report Packages, or shared with other users. Dates are able to be saved relative to the current period (e.g., reflect January 2018 to the latest month).

Notifications are displayed to make you immediately aware of data concerns or updates that are important to call out.

In future releases, this feature will expand to encompass data validation alerts, custom or automated 'significant event' notifications, as well as operational information from the application.
