Troubleshooting

Use this page to problem solve issues you are having with PriceTek Deposits. If you do not find an answer to your issue/problem, please do not hesitate to contact Novantas for support: pricetek_support@novantas.com.

 

System Requirements to run PriceTek Deposits
   

System Component

 PriceTek Requirement
Software Requirements
Operating System Windows 7, Windows 10, or later
Microsoft Office Microsoft Office 2007 or later
Internet & Browser Requirements
Internet Connection Broadband Internet Connection, 5 Mbps or faster
Browser Versions

Within the latest 2 supported versions of:

  • Google Chrome (recommended browser)
  • Internet Explorer or Edge
  • FireFox
Hardware Requirements
Screen Resolution 1280+ x 900+
PC Hardware Capable of running the above 3rd party software

 

These reflect PriceTek’s latest requirements. However, older configurations deemed compatible in SOWs or other contractual documents will continue to be supported if they differ from the above.

 

Setting up PriceTek Deposits as a Trusted Site when Using Internet Explorer

If you are not able to access PriceTek Deposits using Internet Explorer because it is "blocked", or if you are having trouble accessing downloaded reports that appear blank when opened, do the following:

1.    From the upper right corner of an Internet Explorer browser window, click the Tools icon and choose Internet options (see right).

Client Admin - Your PriceTek administrators; will be able to add & edit users, delete users, change roles, etc.
2.    Click the Security tab.
3.    Click .
4.    Click .
5.    In the Add this website to the zone field, enter https://solutions.novantas.com
6.    Click Add and then click OK.

 

 

Contacting PriceTek Support

You can contact PriceTek Support directly via the PriceTek Deposits tool:

1.    From the upper right corner of any PriceTek Deposits screen, click the Envelope icon .

 
2.    Click the Select reason drop-down and choose either Data Inquiry, Bug Report, Usage Question, or Other.
3.    Then, type your Message and click .

Your message is sent to the entire PriceTek Support Team. Someone from the Support team will contact you directly.
