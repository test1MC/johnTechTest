Using Freeform Analytics

Freeform Analytics allows you to self-explore all underlying datasets.

 

Creating a Freeform Workbook from Scratch
1.    From any analytic Section landing page, click from the blue toolbar at the top of the page.

The Freeform Analytics popup will bring up a selection window displaying all of the Data Sets available in this Analytics module.

2.    Optional: Click View in the Data Dictionary column for any Data Set to review a detailed list of all of the Dimensions (data segmentation field, e.g., Product Group, Pricing Market) and Measures (numeric metrics to analyze, e.g., Balances, Weighted Rate) within that Data Set.
3.    From the list, choose one of the existing Data Sets in this Analytics module.
4.    Click .

Tip: You can also choose Create Workbook from the Data Dictionary list of a Data Set.

The Create Workbook page is displayed (see below).

5.    Drag Dimensions and Measures into the Columns and Rows fields.

The Workbook sheet displays the Dimensions and Measures added to the X-Axis and Y-Axis.

Dimensions: Fields that are used to segment and group the data set. Using multiple Dimensions will break down the data into smaller groups.

  • Click here to view a list of Dimension Names set up specific to your institution and their definitions.

Measures: Numeric metrics that are aggregated across dimensions.

Row or column placement will determine chart style.

6.    Optional: Drag fields into the Filters area to isolate specific data for deeper analysis.
7.    Drag fields into the Marks area to design the visualization.

Color: Colors a chart’s components (i.e., bars or lines) by the field (similar to “group by”). This Mark is similar to the Group By field in PriceTek Deposits.

Size: Size components by the field’s selected aggregation (e.g., Sum, Avg).

Text: Shows the values in text above the visualization (data labels).

Detail: Cuts a visualization by the field (for dimensions only).

Tooltip: Adds the field’s value to an on-hover tool tip.

8.    Optional: Click the Marks drop-down to change the Chart Type (e.g., Area, Bar, Gantt Chart).
9.    Optional: Filters can be added by dragging or right clicking on a dimension.

Filters can be dependent, showing only data that exists after accounting for other selections. Click the triangle to the right of a filter and select only relevant values to activate this behavior.

 

For a deeper dive on how to create Workbooks using Dimensions and Measures, go to Tableau Free Training videos.

Common Use Cases for this Module

 

  • Provide access for advanced users to self-explore underlying data sets and create customized visualizations (Reports, Dashboards, or Stories)
  • Create throw-away workbooks to answer ad-hoc questions, quickly explore unique areas of data

 

Tips, Tricks, & FAQs
  Tips & Tricks

  FAQs  
  I don’t see a dataset that I expect to be available. Help! Datasets will be available in Freeform Analytics within the section most relevant to that type of data. Balance information, for example, will generally be housed within Bank Analytics. Note from the My Analytics section you can access all datasets available within PriceTek.
  Can I build a Dashboard consisting of multiple reports? Yes. Create each report within the same workbook and then create a dashboard from the same workbook.
  Can I export data from Reports? Currently, no. Data can only be exported from Dashboards. Exports from reports will be coming in future releases.
  Can I create custom groupings of dimensions?

Yes, this can be accomplished by using Calculated Fields. For example, if the Product Name dimension has multiple products with a suffix of “_Promo” or “_Standard” to designate type - to create a custom group at that level, Use the following calculated field formula:

IF CONTAINS([Product Name],“_Promo") THEN “Promotion“ ELSEIF CONTAINS([Product Name],“_Standard") THEN "Standard“ ELSE “No Designation” END
  Is it possible to create a data table? Yes, click the Show Me menu and choose the top left option.
  Can I create a dual-axis chart? Can I build a dashboard combining Reports from multiple datasets? Currently, no. However, Novantas Client Services teams have the ability to do these actions outside of the Custom Analytics web interface. Reach out to your CS partners for support here.
  Can I change the number format of my charts? Yes! Click on the bubble of the measure in question in the Columns and/or Rows area and choose Format Number.
  Why does a tile say “No Dashboard”? How do I edit it? This will happen when a report has been created, but no dashboard. This can be edited from the My Analytics section – once editing, drag the report into a dashboard to complete the process.
  How do I create the same visualization easily for each value in a dimension? Drag a dimension into the Pages shelf to enable quick cycling through distinct values.
  How can I tell what data source a visualization references? Every report’s data source is listed at the top of the Data tab, just above Dimensions and Measures.
  My visualization changes type (e.g., bar, line) on its own. How do I stop this? In the Marks shelf, deselect Automatic and select the visualization type you would like to use.
  Can I “undo” an action in Freeform Analytics? Yes! Click the icon in the top left corner to undo the last action.
