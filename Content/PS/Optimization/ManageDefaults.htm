Manage Defaults (Optimization)

You can set Default Rules to streamline the Optimization process.

Note: You can only choose rules that exist - you cannot create rules to set as a default.

 

1.    From the Optimization Inventory screen, click .

The Default Rules popup screen is displayed and current default rules are displayed on the View Defaults tab (see right).

2.    To change your defaults and/or add new ones, click the Change Defaults tab.

3.    Click the drop-down at the top of the Change Defaults tab and choose the Optimization rule to display (e.g., Macro Rules, Competitor Rate Rules, Bound Rules, Price Ordering Rules).

A list of existing rules are displayed below the Search bar (you can only add existing rules as a default.

4.    If a default rules already exists and you want to replace it: Click and choose Remove.
5.    Optional:Use the Search bar to find the existing rule.
6.    To add a rule, click
7.    When finished changing your optimization defaults, click Close.
