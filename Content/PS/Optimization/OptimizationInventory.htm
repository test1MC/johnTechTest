Optimization Inventory

The Optimization module leverages the scenario engine to identify optimal portfolio pricing. Rather than run a series of scenarios using the Scenario Builder, find optimal rates that most efficiently achieve your target under a set of business constraints and based on a set of assumptions that you control that return an optimization pricing grid.

From the Optimization landing page, you can:

  • View Optimizations created by any member of your team
  • Manage existing optimizations you created
  • Manage Defaults (Optimization)
  • Build new optimizations either from scratch or based on an existing optimization

Tips, Tricks, & FAQs
  Tips & Tricks  
  Use the Search bar to find optimizations. Any part of any word in the Optimization name, Creator name, or Description is searchable.
  FAQs  
  When should I use the Optimization functionality and when should I use Scenarios?

Use Scenarios to explore a range of possibilities, for example to see what the potential impact would be of a rate change of +5, 10, 15, 20, OR 25 BPS - and evaluate the changes at each step, and observe the fluctuation across different dimensions.

Use Optimization to identify a single rate position that is optimal given a specific target and constraints.
  How will I know when my Optimization finishes running? You will receive an email once your Optimization finishes running.
  I don't remember if I reran my Optimization after I made some changes. How can I tell? Any Optimization that has changed since the last time it has been run will be flagged with a Gold Changed Since Last Run tag. Additionally, if an Optimization is blue in the inventory (or white in Optimization Analysis), then the results are up to date with the most recent rule changes.
  In my Optimization Inventory, I only see a button to View Output. How do I rerun my Scenario? Click the button to open the Optimization Details panel. This panel includes a Run button, as well as an Edit and Delete button. From here, you can also see the list of rules for this Optimization. Click New Optimization Based On to create a new optimization based on this one.
