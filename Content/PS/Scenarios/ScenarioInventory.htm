Scenario Inventory

From the Scenario landing page, you can:

  • View scenario outputs created by any member of your team
  • Manage existing scenarios you created
  • Build new scenarios either from scratch or based on an existing scenario

Tips, Tricks, & FAQs
  Tips & Tricks  
  Use the Search bar to find scenarios. Any part of any word in the Scenario name, Creator name, or Description is searchable.
  FAQs  
  When should I use the Scenario functionality and when should I use Optimizations? Use Scenarios to explore a range of possibilities, for example to see what the potential impact would be of a rate change of +5, 10, 15, 20, OR 25 BPS - and evaluate the changes at each step, and observe the fluctuation across different dimensions. Use Optimization to identify a single rate position that is optimal given a specific target and constraints.
  How will I know when my Scenario finishes running? You will receive an email once your Scenario finishes running.
  I don't remember if I reran
my Scenario after I made some changes. How can I tell? 		Any Scenario that has changed since the last time it has been run will be flagged with a Gold Changed Since Last Run tag. Additionally, if a Scenario is blue in the inventory (or white in Scenario Analysis), then the results are up to date with the most recent rule changes.
  In my Scenario Inventory, I only see a button to View Output. How do I rerun my Scenario? Click the button to open the Scenario Details panel. This panel includes a Run button, as well as an Edit and Delete button. From here, you can also see the list of rules for this Scenario. Click New Scenario Based On to create a new scenario based on this one.
