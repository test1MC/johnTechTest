Scenario Building Best Practices

 

When creating a scenario for structuring the analysis, there are some best practices to keep in mind:

  • When creating a rate change scenario, only set hypothetical rates on applicable portfolio segments.
    • Only choose the applicable terms in the Rule.
    • Only select the applicable tiers in the Rule.
  • Keep in mind that the scenario predictions may not be as accurate for tiers and markets that have only had limited experience (e.g., high tiers, small markets).
  • When comparing Scenario results, limit actual performance to portfolio segments that are in-scope for the Scenario.
    • Do not consider products that are not included for modeling.
    • For term products, be aware of the maturing balances used to apply the model predictions.
  • Using the curve viewer, examine the Scenario-level elasticity curves used to calculate the Scenario predictions.
    • Determine which cells have strong price-balance relationships and which do not.
    • Investigate if the tested price change falls within the tested historical rate range.
  • If the scenarios were originally run far in advance of the rate change, run the scenarios again in the week of the rate change to ensure that predictions have not changed substantially with movements in the market.
