Reports

The Reports module allows you to create Report Packages that includes a collection of stock reports, custom reports, as well as views from PriceTek's analytic modules that you or your team have "favorited". Once Report Packages are created, you can run and download them - and/or save them and schedule them to be run at a specific time each day, week, or month.

Using the Report Package Builder tab

 

Use the Report Package Buildertab to create Report Packages to run immediately, or save for future or recurring scheduled runs.  Report Packages can contain either stock reports, custom reports, or saved Favorites from the Analytic modules.  The full output from a Report Package will be delivered in a single Excel file, with each Report or Favorite making up one worksheet within the file.

  • Create a Report Package using stock reports, custom reports, and/or "Favorites"
  • Run a Report Package to see immediate output
  • Save a Report Package or Reports for scheduling or future use

Creating a Report Package

To build a package of reports, do the following:

1.    In the left pane, choose a report set to add to the report package:
A.   From the Reports tab, choose a either a Stock or Custom report set.

Use the Stock and Custom checkboxes to filter the report list.

Or,

B.   From the Favorites tab, choose a report that you or your team have created and previously "favorited."

Tip: Use the Search bar to find a report, if necessary.

Note: If a Report is selected, there may be additional parameters to select before you can add the selection to the Report Package. The required parameters will be displayed in the center pane. Favorites will never have additional parameters to add.
2.    If required: From the center pane, choose the dimension(s) you would like to use to create the Report.  

Note: When making multiple selections, values for each item will be added to the Report Package individually.

3.    Click .

The report is added to the Package list in the right column.

4.    Repeat Steps 1 - 3 as needed. Note there is a maximum limit of 25 items in a Report Package.
5.    From the right pane, you can change the order of the Reports as they will appear in your output workbook: click, drag, and drop a report in the final package list to adjust ordering.
6.    Click or .

Choose to run the Report Package immediately for a one time download (available on the Output tab of the Reporting section).

Choose to save the Report Package for future re-use or automated scheduling. If saving the Report Package:

A.   Type the Report Package Name.
B.   Choose your Save Options and click Save again.

Your Report Package is saved to your Report Package Manager module and accessible from there to run or schedule.

 

Using the Report Package Manager tab

 

The Report Package Manager tab displays those Report Packages that have created by any member of your team. Use the Report Package Manager tab to do the following:

  • Run a Report Package
  • Create a report based on an existing Report Package built by you or someone on your team
  • Schedule a Report Package to be run weekly, monthly, or daily (or modify the schedule)
  • Delete a Report Package previously saved

 

Running a Report Package

To run an existing Report Package immediately, click Run Package.

 

Create a Report Package based on an existing one

To create a new Report Package based on one already created by your or someone on your team, click New Based On for the desired Report Package to copy.

1.    For the desired Report Package to copy, click New Based On.

The Report Package Builder tab is displayed, and the collection of reports are already added to your new Report Package.

2.    Edit the collection of reports as needed. Refer to the Reports drop-down section above.

 

Scheduling a Report Package

1.    For any saved Report Package listed on the right side of the module, click Schedule Package.

The Schedule Package popup window is displayed.
2.    Click the Run drop-down arrow and choose Weekly, Monthly, or Daily.
3.    If you chose Weekly: click the On drop-down arrow and choose the day (Monday through Friday).

If you chose Monthly: click the On drop-down arrow and choose the day (1st - 31st).

Tip: To schedule a report package to run on the last day of the month, choose 31st. For months with less days than 31, the report will run on the last day of that month.

4.    Click the At drop-down arrow and choose the time (in 30 minute increments, from 8:00 AM to 7:00 PM).
5.    Click Schedule.

The Report package will be listed under Scheduled Packages list on the right side of the page.

6.    To modify the schedule: click Modify Schedule.

 

Deleting a Report or Package of Reports

To delete a Report Package from the Saved Report Packages list on the right side, click .

To delete a package from the schedule (but not delete the saved Report Package itself), go to the Scheduled Packages list on the bottom left side and click .

 

Using the Completed Report Packages tab

 

The Completed Report Packages Package Manager tab displays those Report Packages that have been run or are in the queue.

 

Downloading a Report Package

You can only download reports after they have been run.

From the Completed Report Packages tab, click Download for the report you wish to access in Excel. Note that completed Report Packages are available for one time download only.

The report is downloaded to your default Downloads folder.
